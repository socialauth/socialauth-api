socialauth-api by socialauth

  • API for websites which themselves connect to external third party APIs using Spring-Social.
  • Acts as an OAuth API, serving user profiles from Spring-Social datastore via JSON/REST
  • Allows custom services to be exposed via JSON/REST quickly and easily
  • Delegates API authentication and security to third party apis using spring-social-security, removing the need to provide own user account management or OAuth client management.
  • Can be connected to easily by Spring-Social clients, using custom spring-social-socialauth modules.
